# 常用功能 (common)

::: tip 提示
本章介绍的功能都由 koishi-plugin-common 插件提供。
:::

koishi-plugin-common 包含了一些基本插件，它们在你使用 `koishi` 库时是默认安装的。

## 指令列表

以下是这个插件包含的指令列表。注明“需要数据库”的指令在没有配置数据库时默认不注册。

| 指令 | 默认权限 | 需要数据库 |
|:-:|:-:|:-:|
| admin | 4 | 是 |
| broadcast | 3 | 是 |
| contextify (ctxf) | 3 | 是 |
| echo | 3 | 否 |
| exec | 4 | 否 |
| exit | 4 | 否 |
| help | 0 | 否 |
| info (i) | 0 | 是 |

你也可以在安装插件时显式地声明某些指令不注册：

```js koishi.config.js
module.exports = {
  plugins: [['common', {
    exit: false,        // 不注册 exit 指令
  }]],
}
```
