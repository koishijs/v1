# 官方插件

Koishi 官方提供了许多插件，它们被分散在了多个仓库中：

# 核心插件

核心插件作为 koishi 的依赖，由 koishijs/koishi 仓库提供，会在本栏有详细的介绍：

名称 版本 功能
plugin-common 常用功能
plugin-nlp 自然语言处理
plugin-recorder 存储聊天记录
plugin-schedule 设置计划任务
plugin-teach 教学功能

# 独立插件

另外一些插件较为复杂，它们将各自拥有自己的仓库，同样在本栏会有介绍：

名称 版本 功能
plugin-chess 下棋
plugin-github-webhook 接入 GitHub Webhook
plugin-gitlab-webhook 接入 GitLab Webhook
plugin-image-search 搜图
plugin-status 状态监控
plugin-monitor 直播监控推送
plugin-turing 接入图灵机器人

# 其他小插件

最后，koishijs/plugins 仓库还提供了大量小插件。它们的文档会直接写在仓库中。

名称 版本 功能
plugin-latex 渲染 LaTeX
plugin-magi 调用 Magi
plugin-maya 计算玛雅日历
plugin-mcping 获取 Minecraft 服务器信息
plugin-oeis 调用 OEIS
plugin-qrcode 生成二维码
plugin-roll 掷骰子
plugin-vm 执行 JavaScript
plugin-wolfram-alpha 调用 WolframAlpha
plugin-youdao 使用有道翻译
